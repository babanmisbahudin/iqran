Kebijakan Privasi

iQran — Aplikasi Al-Qur'an  |  Berlaku sejak 1 April 2026

Aplikasi iQran dikembangkan oleh Baban Misbahudin sebagai aplikasi Al-Qur'an pribadi untuk muraja'ah harian. Kebijakan privasi ini menjelaskan informasi apa yang kami kumpulkan, bagaimana kami menggunakannya, dan hak-hak Anda sebagai pengguna.

1. Informasi yang Dikumpulkan

iQran tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi Anda ke server kami.

Semua data disimpan secara lokal di perangkat Anda, meliputi:

  • Posisi bacaan terakhir (surah dan ayat)
  • Riwayat bacaan harian dan target ayat
  • Daftar surah yang di-bookmark
  • Pengaturan aplikasi (ukuran font, tema, pilihan qari)
  • Cache data Al-Qur'an untuk penggunaan offline

Data ini hanya tersimpan di perangkat Anda dan tidak dapat diakses oleh siapa pun selain Anda.

2. Izin yang Diminta

  • Internet — Untuk mengambil data Al-Qur'an dan streaming audio murottal dari API publik equran.id.
  • Lokasi (GPS) — Hanya digunakan untuk fitur arah kiblat dan jadwal shalat. Data lokasi tidak dikirim ke server mana pun.
  • Wake Lock & Foreground Service — Agar audio murottal tetap berjalan saat layar mati atau aplikasi diminimalkan.

3. Layanan Pihak Ketiga

Aplikasi menggunakan layanan berikut untuk mengambil konten Al-Qur'an:

  • equran.id — API publik untuk teks Al-Qur'an (Arab, latin, terjemahan)
  • cdn.equran.id — CDN audio murottal per surah
  • Google Fonts — Font tipografi (diunduh saat pertama kali digunakan)

Layanan-layanan di atas memiliki kebijakan privasi masing-masing. Kami tidak mengontrol data yang mungkin dikumpulkan oleh layanan tersebut.

4. Donasi

Fitur donasi dalam aplikasi mengarahkan pengguna ke platform eksternal Saweria. Segala transaksi dan data yang dimasukkan di Saweria tunduk pada kebijakan privasi Saweria, bukan kebijakan ini.

5. Keamanan Data

Karena semua data disimpan lokal di perangkat Anda, keamanan data bergantung pada keamanan perangkat Anda sendiri. Kami menyarankan untuk menggunakan kunci layar pada perangkat Anda.

6. Privasi Anak-anak

Aplikasi iQran tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak di bawah usia 13 tahun. Aplikasi ini aman digunakan oleh semua usia karena tidak ada registrasi akun atau pengiriman data pribadi.

7. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan diinformasikan melalui pembaruan aplikasi di Google Play Store. Tanggal "Berlaku sejak" di bagian atas halaman ini akan diperbarui sesuai.
